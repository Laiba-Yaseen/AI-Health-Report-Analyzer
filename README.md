# 🩺 AI Health Report Analyzer & Doctor Recommendation System

## 📌 Overview

The **AI Health Report Analyzer & Doctor Recommendation System** is an AI-powered healthcare automation project that extracts, analyzes, and interprets medical reports using **Optical Character Recognition (OCR)** and **Generative AI** technologies. The system transforms unstructured medical reports into structured health insights, identifies abnormal laboratory values, and recommends the most relevant medical specialists.

This project demonstrates the practical application of **Artificial Intelligence, Healthcare Analytics, Document Processing, and Automation**, making it an ideal portfolio project for AI Engineers and Machine Learning practitioners.

---

## 🎯 Project Objective

The objective of this project is to automate the analysis and interpretation of medical reports, enabling users to understand complex healthcare information quickly and efficiently through AI-powered insights and recommendations.

---

## ✨ Key Features

* 📄 Upload medical reports in **PDF** or **image** format
* 🔍 Extract medical information using **Tesseract OCR**
* 🧠 Analyze reports using **Generative AI**
* ⚠️ Detect abnormal laboratory values automatically
* 👨‍⚕️ Recommend appropriate medical specialists
* 📊 Generate structured and human-readable summaries
* 💾 Support optional database storage for report history

---

## 🩺 Medical Parameters Analyzed

The system can analyze multiple laboratory parameters, including:

* Hemoglobin (Hb)
* Blood Glucose
* White Blood Cell Count (WBC)
* Platelet Count
* Creatinine
* Cholesterol
* Additional pathology test values

## 🏗️ System Workflow

1. Upload a medical report (PDF/Image)
2. Extract text using OCR
3. Clean and preprocess extracted data
4. Detect abnormal health parameters
5. Generate AI-based medical explanations
6. Recommend relevant medical specialists
7. Produce structured health summaries
